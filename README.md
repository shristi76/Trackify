# 🧩 TRACKIFY

A simple and responsive **Kanban-style task management app** built .
It helps you organize tasks visually by moving them across different stages.

---

## 🚀 Features

* 📌 Create new tasks
* 🗂️ Organize tasks into columns
* 🔄 Drag and drop tasks between columns
* ❌ Delete tasks easily
* ✨ Clean UI with glassmorphism modal
* 🎯 Smooth hover animations

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
